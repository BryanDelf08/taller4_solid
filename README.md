#Sistema de gestión de pedidos

Este proyecto busca representar la lógica de SOLID en Python.

El sistema en este punto contiene las clases necesarias para una lógica de orden de pedidos en donde se calcula cierto descuento dependiendo el tipo de usuario y aplicando un impuesto general, además genera un reporte de la carcterísticas del pedido en diferentes formatos.

Requisitos
-Python 3.14.3 o superior

Ejecucion
python main.py

Estructura parcial del proyecto

taller4_solid
 └── main.py
